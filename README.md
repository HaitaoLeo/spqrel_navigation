# spqrel_navigation

A light-weight navigation framework with ROS and Naoqi wrappers.

See the wiki for more details.

----

Note for ROS:

link to catkin workspace

$ cd <your_catkin_ws>/src
$ ln -s <spqrel_navigation_dir>/ros/srrg_localizer2d_ros .

compile 

$ cd <your_catkin_ws>
$ catkin_make
